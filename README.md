# 💰 Smart Expense Tracker

An AI-powered expense tracker that automatically categorizes expenses and provides spending insights.

## 🚀 Features
- Add expenses
- Automatic categorization using Machine Learning
- Expense visualization
- Smart spending suggestions

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
